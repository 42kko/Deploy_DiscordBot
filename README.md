This program is a Discord bot that helps with the deployment of 'HowwAbout'.

## Install
```
brew install libdpp
brew link libdpp
```
This program requires the DPP library.

## Setting


## To start
Before starting, make sure to set up the `env` file inside the `config` directory.
```
make
./bin/mybot
```
